#App 
MosQMedia ist ein Projekt das leider nicht weiterentwickelt wurde. 
Das Projekt ist in der heutigen Türkei ein sinnvolles Tool. 
Viele Menschen besuchen Freitags die Moschee zum Freitags Gebet. 
Wenn man unterwegs ist hält man immer Ausschau zu einer Minarette und versucht dort hin den weg zu finden. 
Die App soll das am nahenden liegende Gebäude finden, User der App die vorher in dieser Moschee waren 
können hier wichtige Notizen hinzufügen, wo kann man parken wo sind die WC oder die Reinigung Räume. 
Man kann die Moschee auf Hygenie und Ausstattung bewerten. Bei Beerdigungen kann die App automatisch 
alle ausgewählten Personen mit einem Klick benachrichtigen, wo Sie den geplanten Tag und welche Moschee 
es stattfinden zugesendet bekommen. Und noch viele Funktionen. Die App ist noch in der Entwicklung und 
daher noch einige Links und das layout ist nicht fertig.

#Installation
#Frontend
React.js im terminal

npm install
danach mit

npm start 
die App starten

#Backend
Laravel im terminal

php artisan serve
